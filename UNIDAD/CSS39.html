Posicionamiento Fijo en CSS

Esta página muestra el uso de position: fixed con dos elementos:

  • Una barra inferior fija (siempre visible en la parte inferior)
  • Una barra lateral derecha fija (siempre visible a la derecha)

Características del posicionamiento fijo

Los elementos con position: fixed se posicionan relativamente a la ventana del navegador, no se mueven al hacer scroll y permanecen en la misma posición.

#barrainferior {
  position: fixed;
  bottom: 0px;
  width: 100%;
  height: 50px;
  background-color: black;
  color: white;
}

#barralateralderecha {
  position: fixed;
  right: 0px;
  top: 0px;
  width: 200px;
  height: 100%;
  background-color: #eee;
}

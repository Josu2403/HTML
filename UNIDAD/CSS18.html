Problema 1: Listas de Lenguajes de Programación

Lenguajes Estructurados

  • Pascal 1970
  • C 1972
  • Fortran 1957
  • COBOL 1959
.lista-estructurada {
  list-style-type: square;
  color: #2980b9;
  background: #e8f4fc;

Lenguajes Orientados a Objetos

  • C++ 1985
  • Java 1995
  • C# 2000
  • Python 1991
.lista-objetos {
  list-style-type: circle;
  color: #27ae60;
  background: #e8f6f3;
  border-left: 4px solid #27ae60;
}

Propiedades CSS aplicadas a las listas:

  • list-style-type: Define el tipo de viñeta (square, circle, decimal, etc.)
  • background: Fondo diferenciado para cada tipo de lista
  • padding: Espaciado interno para mejor legibilidad
  • border: Borde lateral para identificación visual

Problema 2: list-style-position - Inside vs Outside

list-style-position: inside

Los marcadores están dentro del flujo del contenido

  • Pascal - Lenguaje estructurado desarrollado por Niklaus Wirth
  • C - Lenguaje de programación de propósito general
  • Java - Lenguaje orientado a objetos multiplataforma
  • Python - Lenguaje de programación interpretado y multiparadigma
.lista-inside {
  list-style-position: inside;
  list-style-type: decimal;
}

list-style-position: outside

Los marcadores están fuera del flujo del contenido (valor por defecto)

  • Pascal - Lenguaje estructurado desarrollado por Niklaus Wirth
  • C - Lenguaje de programación de propósito general
  • Java - Lenguaje orientado a objetos multiplataforma
  • Python - Lenguaje de programación interpretado y multiparadigma
.lista-outside {
  list-style-position: outside;
  list-style-type: decimal;
  padding-left: 40px;
}

Diferencia entre inside y outside:

  • inside: Los números/viñetas forman parte del contenido y se alinean con el texto
  • outside: Los números/viñetas están fuera del contenido (comportamiento por defecto)
  • Visualmente: Con "inside" el texto se alinea bajo los números, con "outside" el texto tiene su propia línea
  • Espaciado: "outside" generalmente requiere más padding-left para evitar superposiciones
Nota: La diferencia es más evidente cuando los elementos de lista contienen múltiples líneas de texto.

Resumen de Propiedades para Listas

list-style-type

  • disc (por defecto)
  • circle
  • square
  • decimal
  • lower-roman
  • upper-roman
  • none

list-style-position

  • inside
  • outside (por defecto)

list-style-image

Permite usar una imagen personalizada como viñeta:

list-style-image: url('bullet.png');

Los tres pilares de la Web

Tim Berners-Lee es considerado el padre de la Web porque desarrolló los tres elementos básicos para el funcionamiento de la Web:

  • HTML
  • HTTP
  • URL

HTML

HTML, que significa Lenguaje de Marcado de Hipertexto (HyperText Markup Language), es el lenguaje estándar que define la estructura y el contenido de las páginas web. Utiliza etiquetas para organizar elementos como texto, imágenes, enlaces y videos, formando el esqueleto básico que los navegadores web interpretan para mostrar el contenido al usuario. HTML es fundamental para la construcción de sitios web y se complementa con otros lenguajes como CSS para el diseño y JavaScript para la interactividad.

Fuente: HTML, Wikipedia

HTTP

HTTP, o Protocolo de Transferencia de Hipertexto, es el conjunto de reglas que permite la comunicación entre navegadores web y servidores, formando la base de la web para enviar y recibir información como páginas web, imágenes y videos. Funciona bajo un modelo cliente-servidor, donde el navegador (cliente) envía una solicitud al servidor y este responde con el contenido solicitado, y es un protocolo sin estado porque el servidor no recuerda las solicitudes anteriores.

Fuente: HTTP, Wikipedia

URL

Una URL (Localizador Uniforme de Recursos) es la dirección única de un recurso específico en Internet, como una página web, una imagen, un video o un archivo. Funciona de manera similar a una dirección postal para una casa, pero en el mundo digital, permitiendo que los navegadores encuentren y accedan a estos contenidos en la web.

Fuente: URL, Wikipedia

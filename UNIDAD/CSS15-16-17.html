Ejercicios 15, 16 y 17 - Bordes, Padding y Márgenes

Ejercicio 15: Bordes Superior e Inferior

Título Secundario con Bordes Superior e Inferior

Este título utiliza la clase .titulosecundario que define bordes solo en la parte superior e inferior.

.titulosecundario {
  border-top-width: 2px;
  border-bottom-width: 2px;
  border-left-width: 0;
  border-right-width: 0;
  border-style: dashed;
  border-color: #e74c3c;
  padding: 15px;
}

Propiedades aplicadas:

  • border-top-width: 2px - Grosor del borde superior
  • border-bottom-width: 2px - Grosor del borde inferior
  • border-left-width: 0 - Sin borde izquierdo
  • border-right-width: 0 - Sin borde derecho
  • border-style: dashed - Estilo de línea discontinua
  • border-color: #e74c3c - Color rojo para los bordes

Ejercicio 16: Propiedades de Padding

Prueba de Título con Padding Personalizado

Este título utiliza la clase .titulo con padding específico en todas las direcciones.

.titulo {
  padding-left: 20px;
  padding-right: 20px;
  padding-top: 10px;
  padding-bottom: 10px;
  background: #3498db;
  color: white;
  text-align: center;
}

Valores de padding aplicados:

  • padding-left: 20px - Espacio interno izquierdo
  • padding-right: 20px - Espacio interno derecho
  • padding-top: 10px - Espacio interno superior
  • padding-bottom: 10px - Espacio interno inferior

Ejercicio 17: Propiedades de Margen

Título de Primer Nivel

Título de Segundo Nivel

Los títulos anteriores tienen márgenes de 5px en la parte superior e inferior.

h1, h2 {
  margin-top: 5px;
  margin-bottom: 5px;
  padding: 12px;
}

Diferencias clave:

  • Margin: Espacio EXTERNO alrededor del elemento
  • Padding: Espacio INTERNO dentro del elemento
  • Border: Línea que separa el margin del padding
Modelo de Caja CSS: Content → Padding → Border → Margin

Resumen Comparativo

Bordes

Define el contorno del elemento

Padding

Espacio interno alrededor del contenido

Margin

Espacio externo entre elementos

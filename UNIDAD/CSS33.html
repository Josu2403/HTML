Tipos de Cursor CSS

crosshair
Cursor en forma de cruz - usado para selección precisa
default
Cursor predeterminado del navegador
pointer
Mano que indica un enlace clickeable
move
Indica que el elemento se puede mover
e-resize
Flecha que indica redimensionamiento hacia la derecha
ne-resize
Flecha que indica redimensionamiento hacia arriba-derecha
nw-resize
Flecha que indica redimensionamiento hacia arriba-izquierda
n-resize
Flecha que indica redimensionamiento hacia arriba
se-resize
Flecha que indica redimensionamiento hacia abajo-derecha
sw-resize
Flecha que indica redimensionamiento hacia abajo-izquierda
s-resize
Flecha que indica redimensionamiento hacia abajo
w-resize
Flecha que indica redimensionamiento hacia izquierda
text
Cursor de texto (I-beam) para seleccionar texto
wait
Reloj de arena que indica que el sistema está ocupado
help
Cursor con signo de pregunta para ayuda
auto
El navegador decide el cursor apropiado automáticamente

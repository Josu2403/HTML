Unidad 1: Fundamentos del Desarrollo Web

1. Conceptos Generales del Desarrollo Web

¿Qué es el Desarrollo Web?

Es el proceso de crear y mantener sitios o aplicaciones que funcionan en un navegador web.

Frontend vs. Backend:

  • Frontend (Cliente): Todo lo que el usuario ve e interactúa directamente
  • Backend (Servidor): La lógica "detrás de escena" que gestiona los datos
  • Full Stack: Desarrollador que trabaja en ambos lados

Ejemplo: La Analogía del Restaurante

Frontend: El menú, la decoración, la mesa y la comida presentada.

Backend: La cocina, los chefs, las recetas y el inventario.

2. Entornos de Desarrollo Web

El Taller del Desarrollador

  • Editor de Código: VS Code, Sublime Text, Atom
  • Control de Versiones: Git con GitHub o GitLab
  • Herramientas de Diseño: Figma, Adobe XD
  • Navegador Web: Para testing y debugging

Ejemplo: Flujo de Trabajo con Git

git init
git add .
git commit -m "Añadido formulario de contacto"
git push origin main

3. Configuración de Entornos de Desarrollo

Poner a Punto el Taller

  • Instalar VS Code + extensiones útiles
  • Configurar servidor local (XAMPP, WAMP, Live Server)
  • Instalar entorno de ejecución (Node.js, PHP, Python)

Ejemplo: Configuración con XAMPP

Paso 1: Descargar e instalar XAMPP

Paso 2: Iniciar Apache y MySQL

Paso 3: Guardar archivos en carpeta htdocs

Paso 4: Acceder via http://localhost/tu_proyecto

<?php
echo "¡Mi primer sitio con XAMPP!";
?>

Unidad 2: HTML - Formularios y Tablas

1. Formularios HTML

¿Qué son los Formularios HTML?

Permiten a los usuarios introducir datos que pueden ser enviados a un servidor.

  • Registro de usuarios
  • Inicios de sesión
  • Formularios de contacto
  • Búsquedas y filtros

Ejemplo: Estructura Básica

<form action="/procesar.php" method="POST">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <button type="submit">Enviar</button>
</form>

Mis Formularios Creados:

  • Formulario de Registro - Formulario completo con validación
  • Formulario de Contacto - Formulario simple con campos de mensaje
  • Formulario de Login - Interfaz de inicio de sesión

2. Tablas HTML

¿Qué son las Tablas HTML?

Permiten organizar datos en filas y columnas.

  • Mostrar datos tabulares
  • Comparar información
  • Crear horarios o calendarios
  • Presentar datos estadísticos

Ejemplo: Estructura Básica

<table border="1">
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
        <th>Ciudad</th>
    </tr>
    <tr>
        <td>Ana</td>
        <td>25</td>
        <td>Madrid</td>
    </tr>
</table>

Mis Tablas Creadas:

  • Tabla de Productos - Información de productos y precios
  • Tabla de Horarios - Horario semanal de actividades
  • Tabla de Estudiantes - Lista de estudiantes con calificaciones

Ejercicio 14: Propiedades de Bordes

Título Nivel 1 - Borde Sólido

Este título utiliza: border-style: solid con color azul (#0000ff) y 2px de ancho.

Título Nivel 2 - Borde Punteado

Este título utiliza: border-style: dotted con color azul (#0000ff) y 2px de ancho.

Título Nivel 3 - Borde Doble

Este título utiliza: border-style: double con color azul (#0000ff) y 2px de ancho.

Propiedades CSS utilizadas:

  • border-width: Define el grosor del borde (2px)
  • border-style: Define el estilo del borde (solid, dotted, double)
  • border-color: Define el color del borde (#0000ff - azul)
